# App-Eventos-INATEL

Aplicativo web protótipo para explorar eventos acadêmicos e tecnológicos da INATEL, com uma experiência visual voltada para mobile e navegação simples entre telas.

## Objetivo do projeto

Este projeto foi desenvolvido como uma interface interativa para apresentar eventos, permitir salvar favoritos e facilitar a navegação entre detalhes, início e lista de eventos. A proposta é oferecer uma experiência intuitiva para quem deseja acompanhar oportunidades acadêmicas e tecnológicas de forma prática.

## Landing page

A landing page foi adicionada como uma etapa extra de apresentação do projeto. Ela fica fora do escopo inicial do protótipo, mas foi criada para deixar a ideia mais profissional, organizada e atrativa para quem visualizar o projeto pela primeira vez.

Sua função é apresentar o propósito da aplicação, destacar os principais pontos do conceito e servir como porta de entrada antes do usuário acessar o protótipo principal.

## Tecnologias

- HTML
- CSS
- JavaScript
- SVG inline para ícones locais
